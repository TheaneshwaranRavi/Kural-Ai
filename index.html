Features
Everything you need to study, independently
100% voice operated
No screen, no typing. Speak in Tamil or English. Adjustable speed, pause anytime, repeat any answer.
Spaced repetition
SM-2 algorithm tracks mastery per topic. Schedules review at optimal time. Adapts to your learning pace.
7 RAG pipelines
Naive, Corrective, Self-RAG, Fusion, Graph, HyDE, and Agentic RAG for every question type.
Full TN syllabus
Class 6–12 textbooks, TNPSC syllabi, previous year papers — all indexed and voice-searchable.
Daily current affairs
Scraping agent fetches exam-relevant news every morning. Auto-summarized. Delivered as 5-minute audio brief.
Offline first
LLM runs fully on-device. Zero internet needed after setup. Works on basic laptops with 8GB RAM.
